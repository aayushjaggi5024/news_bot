This is a responsive telegram bot that can we used for news and small talks
the bot has 8 different news categories and has respective news tablets for each category

you can read news of one of "8 categories,of any country, in any language"(languages as per gnewsclient)

Varoius tech used are:
python
gnewsclient : to extract latest news from web
google dialogflow : to train model
varoius other python libraries like telegram.ext, logging ,dialogflow_v2

marksheets styles have been used for categories tablets
you can check the interface of bot from images provided 

Bot API was request from telegram app(will not be with the code as anyone can misuse it)

also you can have general small talks with the bot as it is quite trained for such things
